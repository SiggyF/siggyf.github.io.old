Overview of packages

Tappy

  • Author: S. M. Barbosa & Tim Cera
  • Last version: 2012
  • Tests: unit tests
  • Used in publications: <10
  • Citation: Geophysical Research Abstracts, Vol. 11, EGU2009-5154, 2009 EGU General Assembly 2009

Features

  • Irregular time series
  • IHO convention for constituents
  • > 1 year
  • Filters: doodson, usgs, fft, boxcar

Install

Download at

http://sourceforge.net/projects/tappy/files/tappy/tappy-0.9.0/tappy-0.9.0.tar.gz

pip install tappy-0.9.0.tar.gz

Usage

              tappy.py  analysis data.txt data_def.txt

data.txt


Station Date       Time      WL    Sigma I L 
8720220 01/01/2000 00:00   0.563   0.008 0 0
8720220 01/01/2000 01:00   0.204   0.008 0 0
8720220 01/01/2000 02:00  -0.050   0.005 0 0

data_def.txt


  decimal_sep = "."
  parse = [ 
  integer_as_string('station', exact=4),
  positive_integer('day'),
  positive_integer('month'),
  positive_integer('year'),
  positive_integer('hour'),
  positive_integer('dummy'),
  real('water_level'),
  ]
